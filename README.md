# Prospera Marketing - Landing Page

Este es el proyecto de la Landing Page principal de Prospera Finanzas.

## Tecnologías
- Vite + React + TypeScript
- Tailwind CSS 4

## Instalación
1. `npm install`
2. `npm run dev`

## Despliegue en Vercel
Este proyecto debe estar conectado al dominio `prosperafinanzas.com`.
Asegúrate de configurar los DNS en Namecheap para apuntar a Vercel.
